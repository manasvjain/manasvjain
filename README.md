</summary>

### Engineering Highlights

| Category | Details |
|----------|---------|
| **Stack** | Python • PyTorch • Node.js • Express • MongoDB • Hugging Face • GitHub Actions |
| **Scale** | 2.9 Million Parameter Transformer |
| **Performance** | 87% Accuracy • 0.86 Macro F1 Score |
| **Security** | REST APIs • MongoDB Validation • Secure Deployment |
| **Impact** | Accelerates genomic mutation analysis through AI |
| **Repository** | https://github.com/manasvjain |

### Project Overview

- Designed an encoder-only Transformer architecture.
- Developed FFT-inspired DNA preprocessing.
- Built end-to-end prediction APIs.
- Integrated Hugging Face Spaces deployment.
- Automated CI/CD using GitHub Actions.
- Stored prediction history using MongoDB.
- Built an interactive dashboard with genomic visualizations including GC/AT content and nucleotide frequency.
